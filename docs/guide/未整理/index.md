### 地图遇到数据不显示时

```
属性查询一般使用EPSG:4326
获取wms源(栅格raster)使用EPSG: 3857 (为了在web端地图上展示, mapboxgl就是只遵循3857, 不支持4326)
EPSG:3857（投影）：数据的可读性差和数值大存储比较占用内存。
EPSG:4326（地理）：使用此坐标系会导致页面变形。
通常：数据存储在EPSG:4326中，显示在EPSG:3857中
```
